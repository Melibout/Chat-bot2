# Assistant Virtuel pour Itinéraires de Ville en Ville 

Ce script Python utilise l'API OpenAI pour créer un assistant virtuel capable de fournir des itinéraires de ville en ville. L'utilisateur peut préciser sa ville de départ et sa destination, puis demander à l'assistant de générer un itinéraire. 

## Configuration 

1. Assurez-vous d'avoir une clé API OpenAI valide pour pouvoir accéder aux fonctionnalités de l'API. 
2. Installez la bibliothèque `openai` en exécutant `pip install openai`. 

## Utilisation 

1. Exécutez le script en Python. 
2. Suivez les instructions pour préciser la ville de départ et la destination. 
3. L'assistant demandera les informations nécessaires pour générer un itinéraire. 
4. Consultez la réponse de l'assistant pour obtenir l'itinéraire proposé.
